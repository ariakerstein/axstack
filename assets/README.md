# Creating demo GIFs

## fundraise-demo.gif

1. Use [Kap](https://getkap.co/) (Mac) or similar
2. Open Claude Code with a deck project
3. Run `/fundraise review path/to/deck.html`
4. Record ~10-15 seconds showing the scorecard output
5. Export as GIF (800px max width, 15fps)
6. Save as `fundraise-demo.gif`

## Tips

- Keep under 15 seconds
- Show the scorecard clearly
- Optimize with `gifsicle -O3` if over 5MB
